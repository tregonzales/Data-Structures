Import the files, common.txt, test1.txt, test2.txt, test3.txt into project folder of eclipse.  Run the program through eclipse.  Then use terminal to guide to the directory containing the classes and input files.  The output files will also be in that directory and those will have the keywords from the input files.  You can then put these output files into a text editor to view the keywords.  The main method will also print out the keywords derived from the input files.
